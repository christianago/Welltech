  • Call Debug:

  • Radius Debug:

2. Use EasyLogV3 to trace debug log:

Click "Setting" and key-in the Proxy IP and user ID & password
 
  Click "Action > Connect"  
   
     
3. Search log by ANI (calling number), DNIS (called number) and Call-ID  
  4. Check failed reason code (Refer V2.1 User guide: Appendix 3: WS6500 Status Code)  
     
     
     
     
   
II. Use Wireshark to trace capture:
   
  1. Download & Install Ethereal software > http://www.wireshark.org/download.html
  2. Wireshark capture on both inside NAT and Public network
3. Click "Capture > Option"
 
   
4. Capture Filter: keyin Capture IP (ex. host 192.168.19.17 or 192.168.19.92)
 
   
5. You can see the capture:
 
6. Search log by User ID, ANI (calling number), DNIS (called number) and Call-ID  
  7. Check failed reason code (Refer V2.1 user guide: Appendix 3: WS6500 Status Code)
