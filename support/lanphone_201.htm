Firmware Upgrade Tip:
  • 2m Program: Upgrade all software in VoIP device, including Boot program, Application 
    program…etc. User can upgrade this when there is more than one section of software 
    need to be upgraded. Please be careful because damage of Boot program will cause 
    device to crash forever.
  • Application Program: Application Program is most frequently updated software section, 
    we will suggest customer to upgrade this program in most cases.
  • Boot Program: Upgrade boot section software. User can upgrade this program only in 
    some special cases. Please be careful because damage of Boot program will cause 
    device to crash forever.
